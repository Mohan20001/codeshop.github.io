Syntax

HTML is a Mark up language which discribe documents structure.

  • HTML stand for hyper text mark up language.
  • It contains a set up Markup tags
  • A html startingTag start and end with (< )and(>) respectively.
  • Ending Tag start and end with ( < / ) and (>) respectively.
  • Tags are discribe different document content.
  • Elements contain a Starting Tag,content and Ending Tag.
  • NOTE:some elements doesn't have ending tags

Code:

		        <!DOCTYPE html>
		        <html lang="en">
	                <head>
		             <title>Page title</title>
		         </head>
		           <body>	
		               Hello world!							
		           </body>
		         </html>

Output:

Hello world!
